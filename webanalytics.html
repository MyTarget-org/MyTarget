Web-аналитика

Электронная почта: contactus@mytarget.org


Что же такое веб-аналитика? WA — объединение различных подходов, инструментария, методик, целей для превращения пользователей Сети в клиентов. Проводится сбор, обработка, анализ, растолкование сведений о сайтах и их ЦА.

Для чего нужно применять веб-аналитику

Несмотря на кажущуюся простоту, WA ставит задачу — исследовать естественный трафик на сайт, поиск способов его расширения, расширение ЦА, анализ сущности источников трафика. Результаты этого анализа использует СЕО. Аналитики оценивают, какие запросы приводят посетителей на веб-ресурс, откуда люди приходят, сколько времени и на каких разделах они остаются, чем занимаются. При обнаружении, что один канал работает эффективнее, чем другие, аналитики дают рекомендации по корректировке других каналов — им требуется настройка и анализ.

Интернет-агентство MyTarget анализирует, какие запросы люди вводят в поиск, каков объем поискового трафика в общем трафике ресурса, проводит сравнение поведения посетителей из разных источников, время на странице, глубину просмотра, отдачу по целям и пр. Можно сравнить разные поисковики друг с другом, посадочные страницы для всевозможных поисковых запросов, скорректировать семантическое ядро по каждой странице.

Исследование реферального трафика проводится при рассмотрении сайтов, с которых перешли посетители. Аналитик выясняет, каким было поведение пользователей с конкретных веб-ресурсов и реферальных источников. Сведения помогают корректировать работу СЕО-специалистов, когда они будут работать с источниками реферального трафика, размещая статьи, материалы на сторонних ресурсах и пр.

Проводится анализ прямого трафика — переходов через ввод УРЛ сайта, из Избранного, из закладок. Влиять на эти источники невозможно, но именно это исследование дает представление о самых лояльных клиентах, которые помнят адрес веб-ресурса и заходят на него при необходимости, даже не используя поиск.

Счетчики статистики можно настроить так, чтобы автоматически учитывать поисковый трафик по запросам при вводе точного УРЛ или названия сайта, как прямой. Аналитика нужна, чтобы разобраться: приход покупателя на сайт осуществлен из-за успеха работы СЕО-специалиста или по причине лояльности покупателя, который давно знает о бренде. Результаты анализа активности посетителей помогают прийти к выводам о работе веб-ресурса. Цель исследования — оптимизация сайта, чтобы увеличить число целевых действий посетителей.

WA изучает:

  • базовые параметры эффективности — отказы, время на ресурсе, глубину просмотра, откуда пришли, куда вышли посетители;
  • новый и возвратный трафик, их соотношение, эффективность;
  • потоки (пути по сайту);
  • поиск по ресурсу — долю посещений с внутренним поиском, популярные и уточняющие запросы, эффективность посещений с внутренним поиском и без такового;
  • А/В-тестирование — поведение людей на сайте при различных вариантах веб-страниц. По результатам аналитики оставляют вариант, показавший наилучшую эффективность;
  • мультивариантное тестирование — посетители попадают на версию страницы, сформированную автоматически. Максимально оптимизированная страница останется. Но метод не подходит для сайтов с низкой посещаемостью;
  • карты кликов — долевые, тепловые карты — информацию по каждой веб-странице, популярность ссылок в разделах, заметность ссылок, какие элементы неактивны, когда по ним кликают пользователи;
  • просмотр видеозаписей посещений — поведение посетителей, обнаружение сложностей при использовании сайта и пр.

Как оптимизировать рекламную кампанию (РК)

Оптимизацией РК называют анализ эффективности трафика, детализацию каждого проекта, объявлений. К ней относятся:

  • сравнение эффективности отдельных источников рекламы — ЯД, Яндекс.Маркет, Гугл Адвордс и пр. Сравнение с применением РОИ или СПО, коэффициента транзакций, применимых для анализа электронной торговли в интернет-магазинах с возможностью отслеживания факта продаже через сайт. Сравнение по коэффициентам целей — записи на тест-драйв, подписка на новости электронного журнала;
  • анализ эффективности разных объявлений, их групп внутри 1 источника по одним и тем же параметрам;
  • управленческие решения о перераспределении бюджетов для повышения эффективности рекламы — для повышения ставок на самые результативные каналы трафика, отключение неэффективных объявлений или источников трафика.

Сведения статистики применяют в автоматизированных системах, с помощью которых управляют РК, ставками объявлений. WA позволяет составлять отчеты, которые интегрируют с данными по расходам из рекламных систем.

Какие подходы используют для сбора статистических данных

Преимущества анализа логов сайта:

  • отслеживание действий посетителей на ресурсе в деталях — если система основана на счетчиках, то такие сведения доступны в ограниченных количествах или вне доступа;
  • отслеживание прерванных сеансов — посетитель закрыл страницу, не дожидаясь ее загрузки;
  • простота интеграции — не нужна установка дополнительных кодов, чтобы получить анализ из логов сервера.

Преимущества внешних сервисов статистики:

  • нет зависимости от кэширования, статистику собирают через ДжаваСкрипт;
  • доступны технические параметры ПК пользователей, разрешение экрана, цветность, версия флеш, окна браузера и пр.;
  • генерируются и отслеживаются динамические события;
  • можно зафиксировать проигрывание контента звуковых, видеофайлов и пр.;
  • можно отследить карту кликов, получить видеоролик поведения;
  • междоменное отслеживание — удобно для крупных порталов.

Комплексный подход

Целесообразно для полного веб-анализа использовать оба подхода при сборе статистики. Но если вам нужно выбрать одно направление, то отдавайте предпочтение счетчикам, которые входят в состав WA с огромным инструментарием. Инструменты анализаторов логов требуют участия специалистов, которые расширят их функционал своими силами.
